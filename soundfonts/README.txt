Drop a General MIDI SoundFont (.sf2) file into this folder.

When you drag a MIDI file into the DAW, each track is automatically assigned
its own instrument from the SF2 based on the GM program number embedded in the
MIDI file (e.g. piano on track 1, strings on track 2, drums on track 10, etc.).

Recommended free SF2 files:
  - GeneralUser GS  →  https://schristiancollins.com/generaluser.php
  - FluidR3_GM      →  sudo apt install fluid-soundfont-gm  (Linux)
                       or download from https://member.keymusician.com/Member/FluidR3_GM/index.html
  - SGM-V2.01       →  search "SGM-V2 sf2 download"

Place the .sf2 file directly in this folder (e.g. soundfonts/GeneralUser GS.sf2)
and restart the DAW.  No other configuration needed.
